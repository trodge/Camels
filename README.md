# Camels

Open source trading simulator game set in 11th century silk road.

## Getting started

### Prerequisites

cmake, SDL2, Google flatbuffers, sqlite3, Boost (filesystem, multi_index_containers)

### Installing

Clone the repo

```
git clone git@github.com:trodge/Camels.git
```

Build

```
mkdir build
cd build
cmake ../
```

Run

```
cd ..
build/camels.exe
```
